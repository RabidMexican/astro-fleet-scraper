# astro-fleet-scraper
Script to scrape Guild page for fleet totals for AstroEmpires

## Setup
* Copy example.env
* Download chrome-driver https://chromedriver.chromium.org/downloads
* Fill-in the variables in example.env
* Rename example.env to .env
* ```py main.py```
